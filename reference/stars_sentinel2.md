# Sentinel-2 sample tile

Sentinel-2 sample tile, downloaded from
\<https://scihub.copernicus.eu/\> reads the four 10-m bands: B2 (490
nm), B3 (560 nm), B4 (665 nm) and B8 (842 nm)

## Usage

``` r
stars_sentinel2
```

## Format

An object of class `stars_proxy` (inherits from `stars`) of dimension
10980 x 10980 x 4.
